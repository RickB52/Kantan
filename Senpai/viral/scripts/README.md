# Scripts Folder

> Folder chứa script reel theo tuần · template chuẩn 30s và 60s.

---

## 📁 Convention naming

```
scripts/
├── README.md                     # File này
├── template-30s.md               # Template chuẩn 30s — sẽ tạo Phase 0
├── template-60s.md               # Template chuẩn 60s — sẽ tạo Phase 0
├── week-01.md                    # 4 script tuần 1
├── week-02.md                    # 4 script tuần 2
└── ...
```

### Mã reel trong script

`R-{week}-{day}` — ví dụ `R-01-mon`, `R-03-fri`

---

## 📝 Script structure (mỗi reel)

```markdown
## R-XX-XX — [Title hook]

**Pillar:** #[X]
**Length:** [30s | 60s]
**Source lesson:** [link / module]
**Platform:** TT + FB + YT (cross-post)

---

### HOOK (0–3s)
- **Voice:** [câu hook chính]
- **Visual:** [text overlay LỚN, footage]
- **Audio:** [trending? original? volume]

### BODY (3–25s hoặc 3–50s)
[Setup → Reveal → Bài học]

- **Voice:** [voice over]
- **Visual:** [scene by scene]
- **Footage list:** [b-roll cần dùng / cần gen mới]

### CTA (25–30s hoặc 50–60s)
- **Voice:** [CTA cuối]
- **Visual:** [logo + tagline]
- **Caption hook:** [1 dòng caption]

---

### Caption template

**TikTok:**
```
[Hook lại bằng 1 câu]
→ [CTA]
#tiengnhatcongso #senpai #...
```

**Reels FB:**
```
[Hook]
[2-3 dòng giải thích]
🎌 senpai.vn
```

**YouTube Shorts:**
```
Title: [Hook 60 char]
Description: [Link senpai.vn + module]
```

---

### Production checklist
- [ ] Footage gen / reuse
- [ ] Voice over
- [ ] Edit + subtitle
- [ ] Brand check (watermark · logo · cast outfit)
- [ ] Caption + hashtag
- [ ] Schedule 21:00 VN
```

---

## ⚠️ Lưu ý

- File `week-XX.md` viết 1 tuần TRƯỚC khi post (rolling, không viết hết 12 tuần ngay)
- Script đã viết → status: Draft / Reviewed / Production / Posted
- Sau khi post: ghi metric vào file `viral/metrics.md` không cần update lại trong script
