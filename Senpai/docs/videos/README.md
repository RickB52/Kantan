## Cách sử dụng

### Workflow ghép video
1. Các Kling AI scene (live-action với DYNH/ONG_CHON/HONDA) anh tự generate ngoài
2. Import 6 file motion graphics này vào CapCut/Premiere
3. Sequence theo thứ tự scene number
4. Add voiceover (script tại `voiceover-script-m2.2-email-lesson-01.txt`)
5. Add SFX và BGM theo MOOD note trong script gốc

### Spec
- Codec: H.264 (libx264)
- Frame rate: 24 fps
- Pixel format: yuv420p (universal compat)
- Audio: không có (sẽ add sau)

### Re-render nếu cần chỉnh
Code render Python ở `outputs/render_common.py` + `outputs/scene_XX.py`.
Chỉnh nội dung text/timing trong từng file scene rồi run lại.
