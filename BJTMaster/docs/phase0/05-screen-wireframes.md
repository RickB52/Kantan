## Navigation Structure

```
App
├── AuthStack (unauthenticated)
│   ├── Welcome Screen
│   ├── SignIn Screen
│   ├── SignUp Screen
│   ├── ForgotPassword Screen
│   └── Onboarding Flow (3 steps)
│       ├── Step 1: Set Target Rank
│       ├── Step 2: Daily Goal
│       └── Step 3: Diagnostic Mini Test (10 câu)
│
└── MainStack (authenticated)
    ├── BottomTabNavigator
    │   ├── Tab: Home (Dashboard)
    │   ├── Tab: Learn
    │   ├── Tab: Mock Test
    │   └── Tab: Profile
    │
    ├── LearnStack (from Learn tab)
    │   ├── Learn Hub
    │   ├── Keigo Module (A2)
    │   │   ├── Keigo Overview
    │   │   ├── Keigo Situation Drill
    │   │   └── Keigo Quiz
    │   ├── Reading Practice (B3)
    │   │   ├── Reading Selector
    │   │   ├── Reading Question
    │   │   └── Reading Result
    │   ├── Vocab Module (A1)
    │   │   ├── Vocab List
    │   │   ├── Vocab Detail
    │   │   └── SRS Flashcard
    │   ├── Grammar Module (A3)
    │   │   ├── Grammar List
    │   │   ├── Grammar Detail
    │   │   └── Grammar Quiz
    │   └── [Phase 3B] Listening Module (B1)
    │
    └── Modals (overlay)
        ├── AI Explanation Bottom Sheet
        ├── Paywall Bottom Sheet
        └── Achievement Unlock Modal
```
