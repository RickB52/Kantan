## LESSON 1 — 敬語の5分類

### 🎯 Learning Objectives
1. 文化庁「敬語の指針」（2007年）の5分類を理解する
2. 従来の3分類との違いを説明できる
3. 各分類の代表例を最低3つ挙げられる

---

### 📖 Theory

従来、敬語は「尊敬語・謙譲語・丁寧語」の**3種類**とされてきた。
しかし2007年、文化庁（文化審議会答申）は現代の実態に合わせて**5種類**に再分類した。

#### 5分類の全体像

| 分類 | 働き | 敬意の向き | 代表例 |
|------|------|------------|--------|
| **①尊敬語** | 相手・第三者の行為を高める | → 相手/第三者 | いらっしゃる、おっしゃる、なさる |
| **②謙譲語Ⅰ** | 自分の行為をへりくだり、向かう先を立てる | → 向かう先の人物 | 伺う、申し上げる、お持ちする |
| **③謙譲語Ⅱ（丁重語）** | 自分の行為を丁重に述べる | → 話し相手（聞き手） | 参る、申す、いたす、おる |
| **④丁寧語** | 話し相手に丁寧に述べる | → 話し相手 | です、ます、ございます |
| **⑤美化語** | ものごとを美化して述べる | （対象なし） | お酒、お料理、ご飯 |

> **📌 BJTポイント**: 読解・語彙文法問題では①〜③の区別が頻出。特に②と③の混同が最大のトラップ。

---

### 💡 3分類 vs 5分類 — なぜ変わったか

| 旧3分類 | 新5分類 | 変更理由 |
|---------|---------|---------|
| 謙譲語（1つ） | 謙譲語Ⅰ＋謙譲語Ⅱ | 「伺う」と「参る」は働きが根本的に異なる |
| 丁寧語（1つ） | 丁寧語＋美化語 | 「ございます」と「お酒」は性質が異なる |

---

### 📝 例文で確認

```
【状況】山田さん（自分）が課長（上司）に、田中先生（第三者）を訪問すると伝える場面

❌ 3分類だけで考えた場合：
「田中先生のところに参ります」→「謙譲語だからOK」（曖昧）

✅ 5分類で正確に考えた場合：
「田中先生のところに参ります」
→ 「参る」は謙譲語Ⅱ
→ 課長（話し相手）に丁重に述べている
→ 田中先生を「立てている」わけではない ← これが重要！

「田中先生のところに伺います」
→ 「伺う」は謙譲語Ⅰ
→ 田中先生（向かう先）を立てている
→ 両方とも正しいが、意味が違う
```

---

### ✏️ Exercises

**Exercise 1-A：分類判定（4択）**
次の下線部の敬語の種類を選んでください。

1. 部長が**いらっしゃいます**。
   → ① 尊敬語　② 謙譲語Ⅰ　③ 謙譲語Ⅱ　④ 丁寧語

2. 明日、御社に**伺います**。
   → ① 尊敬語　② 謙譲語Ⅰ　③ 謙譲語Ⅱ　④ 丁寧語

3. 私は田中と**申します**。
   → ① 尊敬語　② 謙譲語Ⅰ　③ 謙譲語Ⅱ　④ 丁寧語

4. こちらに資料が**ございます**。
   → ① 尊敬語　② 謙譲語Ⅰ　③ 謙譲語Ⅱ　④ 丁寧語

**解答**: 1→① 2→② 3→③ 4→④

---

**Exercise 1-B：BJT形式（空欄補充）**
（　）に入る最もよいものを選んでください。

> 「お客様は今、会議室に（　　　）。」
> 1. おります　2. いらっしゃいます　3. まいります　4. おっしゃいます

**解答**: 2 — 「お客様」（相手）の存在を高める → 尊敬語「いらっしゃる」

---

### 🌐 Cultural Note
日本のビジネスでは、5分類の正確な知識より**「誰を立てているか」という意識**が重要。
敬語を間違えると「あの人は社外の人を立てない」「内輪意識が強い」という印象を与えることがある。
BJTはこの「意識」を問う問題が多い。

## LESSON 2 — 尊敬語

### 🎯 Learning Objectives
1. 尊敬語の核心「相手/第三者の行為を高める」を理解する
2. 主要動詞の尊敬語形を正確に言える（不規則変化含む）
3. 「〜れる/られる」との使い分けができる

---

### 📖 Theory

#### 尊敬語の定義（文化庁）
> 「相手側又は第三者の行為・ものごと・状態などについて、その人物を立てて述べるもの」

**核心**: 動作の主体（行為者）が目上の人のとき → 尊敬語

---

#### 主要動詞の尊敬語一覧

| 普通語 | 尊敬語（特別形） | 尊敬語（お〜になる形） | 意味 |
|--------|----------------|----------------------|------|
| いる | **いらっしゃる** | — | いる |
| 行く | **いらっしゃる** | お行きになる | 行く |
| 来る | **いらっしゃる／お越しになる** | — | 来る |
| 言う | **おっしゃる** | — | 言う |
| する | **なさる** | — | する |
| 食べる/飲む | **召し上がる** | — | 食べる/飲む |
| くれる | **くださる** | — | くれる |
| 見る | ご覧になる | — | 見る |
| 知る | ご存知 | — | 知っている |
| もらう | — | お受け取りになる | もらう |

> **📌 注意**: 「いらっしゃる」は「いる・行く・来る」の3つをカバーする。文脈で判断が必要。

---

#### 「〜れる/られる」との違い

| 形式 | 丁寧度 | 使用場面 |
|------|--------|---------|
| 特別形（いらっしゃる等） | ★★★ 高い | フォーマル/対外的 |
| お〜になる | ★★☆ 中〜高 | 標準的なビジネス |
| 〜れる/られる | ★☆☆ 低め | インフォーマル/社内 |

```
❌ 社長が会議室に来られました。（フォーマルな場面では不十分）
✅ 社長が会議室にいらっしゃいました。
✅ 社長が会議室にお越しになりました。
```

---

### 📝 ビジネス例文（BJT頻出場面）

**場面1: 電話対応**
```
「田中部長は、ただいま席を外して（いらっしゃいます / おられます）。
折り返しお電話させていただきます。」
→「いらっしゃいます」が正式。「おられます」は謙譲語Ⅱ「おる」の尊敬用法で
　 地域によっては使われるが、BJT基準では避けるのが無難。
```

**場面2: 来客対応**
```
「部長はただいま会議中でございます。
山田部長はもうすぐこちらにいらっしゃいます。」
```

**場面3: メール書き言葉**
```
「先日ご送付いただいた資料を拝見いたしました。
ご確認の上、ご返信をいただければ幸いです。」
```

---

### ✏️ Exercises

**Exercise 2-A：変換問題**
普通語を尊敬語に直してください。

1. 「社長が言った」→「社長が（　　　）」
2. 「部長が食べた」→「部長が（　　　）」
3. 「お客様が来た」→「お客様が（　　　）」

**解答**: 1→おっしゃった　2→召し上がった　3→いらっしゃった / お越しになった

**Exercise 2-B：BJT形式**
> 「ご不明な点がございましたら、いつでも（　　　）ください。」
> 1. おっしゃって　2. 申し上げて　3. 伺って　4. 参って

**解答**: 1 — 相手（顧客）に「言う」行為を促す → 尊敬語「おっしゃる」

## LESSON 3 — 謙譲語Ⅰ

### 🎯 Learning Objectives
1. 謙譲語Ⅰの核心「向かう先を立てる」を理解する
2. 「お〜する/ご〜する」形の構造を使いこなす
3. 謙譲語Ⅰが使えない相手（立てられない人物）を判断できる

---

### 📖 Theory

#### 謙譲語Ⅰの定義（文化庁）
> 「自分側から相手側又は第三者に向かう行為・ものごとなどについて、
> その**向かう先の人物**を立てて述べるもの」

**核心**: 自分の動作が「向かう先」の人物が目上なら → 謙譲語Ⅰ

---

#### 主要動詞の謙譲語Ⅰ一覧

| 普通語 | 謙譲語Ⅰ | 意味・使用場面 |
|--------|---------|-------------|
| 行く | **伺う** | 目上の人のいる場所へ行く |
| 聞く/尋ねる | **伺う** | 目上の人に聞く/確認する |
| 言う | **申し上げる** | 目上の人に言う |
| もらう | **いただく** | 目上の人からもらう |
| もらう（行為） | **頂戴する** | よりフォーマル |
| 見る | **拝見する** | 目上の人のものを見る |
| 読む | **拝読する** | 目上の人の文書等を読む |
| 知る | **存じ上げる** | 人物について知っている |
| あげる/やる | **差し上げる** | 目上の人に与える |
| 見せる | **お目にかける** | 目上の人に見せる |
| 会う | **お目にかかる** | 目上の人に会う |

---

#### 「お〜する/ご〜する」構造

```
【構造】お + 動詞のます形 + する/いたす
      ご + 漢語動詞（名詞） + する/いたす

【例】
お待ちする → 目上の人を待つ
お届けする → 目上の人に届ける
ご連絡する → 目上の人に連絡する
ご説明する → 目上の人に説明する
```

> **📌 注意**: 訓読み動詞 → 「お〜」、音読み（漢語）動詞 → 「ご〜」が原則

---

### 📝 使えない相手（重要！BJTトラップ）

謙譲語Ⅰは「向かう先を立てる」ため、**立てる必要のない相手**（目下・同僚・自社内の人物）には使えない。

```
❌ 母のところに伺います。（「母」を立てることになる）
✅ 母のところに参ります。（謙譲語Ⅱ：単に丁重に述べる）

❌ 弟に申し上げた。（「弟」を立てることになる）
✅ 弟に言いました。（普通の丁寧語）

❌ 部下にご説明しました。（部下を立てることになる）
✅ 部下に説明いたしました。（謙譲語Ⅱ：丁重に述べる）
```

---

### 📝 ビジネス例文（BJT頻出場面）

**場面1: アポイント**
```
「来週の火曜日、午後3時に御社に伺います。
よろしければ、ご都合をお聞かせください。」
```

**場面2: 資料確認メール**
```
「ご送付いただきました資料を拝見いたしました。
内容について、一点ご確認申し上げたい点がございます。」
```

**場面3: 電話で上司に伝言**
```
（外部の取引先に対して）
「田中はただいま外出しております。
戻り次第、こちらからご連絡申し上げます。」
```

---

### ✏️ Exercises

**Exercise 3-A：適切な謙譲語Ⅰを選べ**

1. 部長に企画書を（　　　）。
   → ①見ました　②拝見しました　③ご覧になりました

2. 先生のご意見を（　　　）たいと思います。
   → ①聞き　②伺い　③おっしゃり

**解答**: 1→② 2→②

**Exercise 3-B：誤りを指摘せよ（BJT表現読解形式）**
> 「明日、母のところに伺います。」
> この文の問題点を説明し、正しい形に直してください。

**解答**:「伺う」は謙譲語Ⅰで「向かう先（母）を立てる」ことになるが、自分の母を立てるのは不適切。正しくは「母のところに参ります」（謙譲語Ⅱ：丁重に述べるだけ）。

## LESSON 4 — 謙譲語Ⅱ（丁重語）

### 🎯 Learning Objectives
1. 謙譲語Ⅱの核心「話し相手に丁重に述べる」を理解する
2. 主要な謙譲語Ⅱ動詞を正確に使える
3. 謙譲語Ⅱが向かう先を問わず使える理由を説明できる

---

### 📖 Theory

#### 謙譲語Ⅱの定義（文化庁）
> 「自分側の行為・ものごとなどを、話や文章の**相手（聞き手）**に対して丁重に述べるもの」

**核心**: 動作の向かう先ではなく、**聞き手（話し相手）**への敬意
→ 向かう先が誰であっても使える

---

#### 謙譲語Ⅱ 主要動詞一覧

| 普通語 | 謙譲語Ⅱ | 補足 |
|--------|---------|------|
| 行く/来る | **参る** | 場所を問わず使える |
| 言う | **申す** | 自己紹介「〜と申します」 |
| する/〜する | **いたす** | 最も汎用性が高い |
| いる | **おる** | 「ております」の形でよく使う |
| 知っている | **存じる** | 「存じます」「存じております」 |
| 思う | **存じる** | 「〜と存じます」 |

---

#### なぜ向かう先を問わないのか

```
「明日、田中先生のところに参ります。」（課長に話している）
→ 「参る」の敬意の向き：課長（聞き手）への丁重さ
→ 田中先生（向かう先）を立てているわけではない

「明日、弟のところに参ります。」（課長に話している）
→ 同じく課長に丁重に述べているだけ
→ 弟を立てていない → 使える！

比較：
「明日、弟のところに伺います。」
→「伺う」= 謙譲語Ⅰ → 弟（向かう先）を立てることになる → 不自然・誤用
```

---

### 📝 ビジネス例文（BJT頻出場面）

**場面1: 自己紹介**
```
「ABCシステムズの山田と申します。
本日はお時間をいただきまして、ありがとうございます。」
```

**場面2: 状況報告（電話・メール）**
```
「現在、担当の田中は出張しております。
来週月曜日に戻る予定でございます。」
```

**場面3: 意見表明**
```
「この件につきましては、来週中に対応いたしたく存じます。
ご了承のほど、よろしくお願い申し上げます。」
```

**場面4: 動作の丁重表現**
```
「後ほど、こちらからご連絡いたします。」
「確認の上、改めてご報告申し上げます。」
```

### ✏️ Exercises

**Exercise 4-A：空欄補充**

1. 「私、佐藤と（　　）します。よろしくお願い申し上げます。」
   → ①おっしゃい　②申し　③伺い

2. 「担当者はただいま外出して（　　　）ります。」
   → ①いらっしゃ　②おられ　③お

   → ①いたし　②なさい　③されてい

**解答**: 1→② 2→③（「おります」） 3→①

**Exercise 4-B：BJT形式**

> 1. おっしゃって　2. 申し上げて　3. 伺って　4. 参って

**解答**: 1 — 「相手（電話の相手）に言ってもらう」→ 相手の行為 → 尊敬語「おっしゃる」

## LESSON 5 — 謙譲語Ⅰ vs 謙譲語Ⅱ【最重要・BJT最頻出】

### 🎯 Learning Objectives
1. 「伺う vs 参る」「申し上げる vs 申す」の違いを確実に区別できる
2. BJTの謙譲語トラップ問題を解けるようになる
3. 混同による誤用パターンを5つ以上指摘できる

---

### 📖 Theory — 決定的な違い

| | 謙譲語Ⅰ | 謙譲語Ⅱ |
|--|---------|---------|
| **敬意の向き** | 動作の向かう先（第三者も含む） | 話し相手（聞き手） |
| **使える相手** | 目上・尊重すべき人物のみ | 誰に向かう行為でも可 |
| **代表語** | 伺う、申し上げる、差し上げる | 参る、申す、いたす、おる |
| **言い換え** | 向かう先が目下なら使えない | 向かう先が目下でも使える |

---

### 🔍 ペア比較（最重要）

#### 「伺う」vs「参る」

```
「田中先生のところに伺います。」
→ 謙譲語Ⅰ：田中先生（向かう先）を立てている
→ 田中先生が目上/尊重すべき人物 → 正しい

「田中先生のところに参ります。」
→ 謙譲語Ⅱ：聞き手に丁重に述べる
→ 田中先生を立てているわけではない → 使える（ただし立てる効果なし）

「弟のところに伺います。」← ❌ 誤用
→ 謙譲語Ⅰ：弟を立てることになる → 不適切

「弟のところに参ります。」← ✅
→ 謙譲語Ⅱ：聞き手に丁重に述べるだけ → 適切
```

#### 「申し上げる」vs「申す」

```
「部長にご報告申し上げます。」
→ 謙譲語Ⅰ：部長（向かう先）を立てている

「私は山田と申します。」
→ 謙譲語Ⅱ：聞き手に自己紹介を丁重に述べる
→ 向かう先（自分）を立てているわけではない

「部下に申し上げました。」← ❌ 誤用
→ 部下（向かう先）を立てることになる → 不適切

「部下に申しました。」← ✅
→ 謙譲語Ⅱ：聞き手に丁重に述べる → 適切
```

---

### 📝 BJT頻出トラップパターン

**パターン1：身内に謙譲語Ⅰを使う誤用**
```
❌「父のところに伺ってきます。」→ 父を立てることになる
✅「父のところに参ってきます。」
```

**パターン2：社内の目下に謙譲語Ⅰを使う誤用**
```
❌「新入社員の鈴木に説明申し上げました。」→ 鈴木を立てることになる
✅「新入社員の鈴木に説明いたしました。」
```

**パターン3：「ご〜する」を誰にでも使う誤用**
```
❌「部下にご説明しました。」→ 部下を立てることになる
✅「部下に説明いたしました。」
```

**パターン4：バスや乗り物に「参る」を使う場合（特殊）**
```
✅「バスが参りました。」← 謙譲語Ⅱの特殊用法。自分側でない対象にも使える。
→ 文化庁も認めている用法（乗り物・天候等）
```

---

### ✏️ Exercises

**Exercise 5-A：正誤判定＋修正（BJT表現読解形式）**

次の文の敬語の使い方は正しいですか。誤りがあれば修正してください。

1. 「明日、母の誕生日なので、母のところに伺います。」
2. 「新人の鈴木に資料をお届けしました。」
3. 「先生に提出書類を差し上げました。」
4. 「来週の月曜日、御社に参ります。」

**解答**:
1. ❌ → 「参ります」（母を立てるのは不適切）
2. ❌ → 「届けました/渡しました」（目下の鈴木を立てることになる）
3. ✅ 正しい（先生=目上への行為）
4. ✅ 正しい（謙譲語Ⅱで丁重に述べる）

**Exercise 5-B：BJT総合問題**
> 上司（山田部長）との会話。社員（自分）が話している。
>
> 「部長、来週の展示会に弊社から3名（　①　）予定です。
> 私も（　②　）予定ですので、当日ご一緒させていただければ幸いです。
> 詳細は後ほど改めてメールで（　③　）いたします。」
>
> ① a.参る　b.伺う　c.いらっしゃる
> ② a.参る　b.伺う　c.いらっしゃる

**解答**:

② a（参る：謙譲語Ⅱ、自分が行く。「伺う」だと展示会を立てることになり不自然）
③ a（ご連絡申し上げ：部長への連絡 → 謙譲語Ⅰ）

## LESSON 6 — ウチ・ソト原則

### 🎯 Learning Objectives
1. ウチ（内）・ソト（外）の概念を理解する
2. 社内の上司を「ソト（外部）」に対してどう扱うかを判断できる
3. 取引先・顧客対応での典型的な切り替えを使いこなす

---

### 📖 Theory

#### ウチ・ソト原則とは
日本語の敬語には「グループ内外」の概念がある。

- **ウチ（内）**: 自分が所属するグループ（自社・自分の家族）
- **ソト（外）**: 相手が所属するグループ（他社・取引先）

**原則**: ソト（外部）に対して話すとき、ウチの人物は低めて扱う

---

#### 具体的な切り替え

| 場面 | 社内での呼び方 | ソト（外部）への言い方 |
|------|-------------|---------------------|
| 自社の部長（田中） | 田中部長 | **田中/弊社の田中** |
| 自社の部長の行為 | 部長がいらっしゃる | **田中はおります** |
| 自社上司の発言 | 部長がおっしゃった | **田中が申しておりました** |
| 自社上司への敬称 | 〜部長、〜さん | **（敬称なし）田中/田中め** ※超フォーマルのみ |

---

#### 頻出場面：電話応対

```
【外部の田中様から、自社の山田部長への電話】

❌ 「山田部長はただいまいらっしゃいません。」
→ 自社の部長に尊敬語 → ソト（田中様）の前でウチを高めてしまう

✅ 「山田はただいま席を外しております。」
→ 自社の部長（山田）をウチとして低めて表現
→ 「おります」= 謙譲語Ⅱ で丁重に述べる
```

---

### 📝 ウチ・ソト切り替え例文一覧

```
【取引先への電話・メール】

❌「田中部長はご存知でしょうか。」
✅「田中は存じておるかと思います。」（田中をウチとして扱う）

❌「弊社の鈴木部長がおっしゃっていたのですが」
✅「弊社の鈴木が申しておりましたが」

❌「課長が確認されてからご連絡します。」
✅「課長が確認いたしましてから、改めてご連絡申し上げます。」
   （行為者が自社内なら謙譲語を使う）
```

---

### ✏️ Exercises

**Exercise 6-A：変換問題**
外部の取引先に電話で話す場面。下線部を正しく直してください。

1.「部長の鈴木は_ただいま会議に_**いらっしゃいます**。」
2.「先ほど、田中部長が_資料をご確認**なさいました**。」
3.「_部長_は今日の会議で_**おっしゃって**いましたが…」

**解答**:
1.「ただいま会議に_おります_」
2.「田中が確認いたしました」
3.「田中は今日の会議で_申して_いましたが」

## LESSON 7 — 敬語の複合形

### 🎯 Learning Objectives
1. 「お〜いたす」「ご〜申し上げる」の構造を理解する
2. 謙譲語Ⅰ＋Ⅱの複合形がなぜ使われるかを説明できる
3. 複合形の適切な使用場面を判断できる

---

### 📖 Theory

#### 「お（ご）〜いたす」= 謙譲語Ⅰ ＋ 謙譲語Ⅱ

文化庁「敬語の指針」では、この形は：
- 「お〜する」（謙譲語Ⅰ）の「する」を「いたす」（謙譲語Ⅱ）に替えたもの
- 「向かう先を立てる」かつ「聞き手に丁重に述べる」の2つの働き

```
「駅で先生をお待ちいたします。」
= お待ちする（謙譲語Ⅰ：先生を立てる）
+ いたす（謙譲語Ⅱ：聞き手に丁重に）
→ 最高丁寧度の表現
```

---

#### 複合形の丁寧度比較

| 表現 | 構成 | 丁寧度 |
|------|------|--------|
| 待ちます | 普通のます形 | ★☆☆ |
| お待ちします | お〜する（謙譲語Ⅰ） | ★★☆ |
| お待ちいたします | お〜いたす（Ⅰ＋Ⅱ） | ★★★ |
| お待ち申し上げます | お〜申し上げる（Ⅰ＋Ⅰ） | ★★★ フォーマル書面 |

---

### 📝 ビジネスでよく使う複合形

**連絡系**
```
ご連絡いたします（ご〜いたす）
ご報告申し上げます（ご〜申し上げる）
お知らせいたします（お〜いたす）
```

**依頼・謝罪系**
```
お願い申し上げます
お詫び申し上げます
ご迷惑をおかけいたしました
```

**確認・送付系**
```
ご確認のほどよろしくお願い申し上げます
資料をお送りいたします
ご査収のほどお願い申し上げます
```

---

### ✏️ Exercise

**Exercise 7-A：適切な複合形を選べ（BJT形式）**

> メールの締めの文：
> 「ご不明な点がございましたら、どうぞ遠慮なくご連絡（　　　）。」
> 1. なさいますよう　2. くださいますよう　3. いただきますよう　4. いたしますよう

**解答**: 3 — 相手に連絡してもらう → 「いただく」（謙譲語Ⅰ）で相手の行為を受ける

## LESSON 8 — よくある誤用 TOP10【BJT直前対策】

### 🎯 Learning Objectives
1. BJTで出題される誤用パターン10種を認識できる
2. 誤用の理由を文法的に説明できる
3. 正しい形に即座に言い直せる

---

### 📖 誤用TOP10

#### ❌ No.1：「〜させていただく」の乱用
```
❌「資料を送らせていただきます。」（許可不要な場面）
✅「資料をお送りいたします。」
→「〜させていただく」は相手の許可や恩恵が前提の表現。
  許可なく一方的に行う行為には使わない。
```

#### ❌ No.2：「ご苦労様」を上司に使う
```
❌「部長、今日もご苦労様でした。」
✅「部長、今日もお疲れ様でした。」
→「ご苦労様」は目上が目下をねぎらう言葉。
```

#### ❌ No.3：「おっしゃられました」（二重敬語）
```
❌「部長がおっしゃられた通りに。」
✅「部長がおっしゃった通りに。」
→「おっしゃる」はすでに尊敬語。さらに「〜られる」をつけると二重敬語。
```

#### ❌ No.4：「拝見させていただきます」（二重謙譲）
```
❌「資料を拝見させていただきます。」
✅「資料を拝見いたします。」
→「拝見」はすでに謙譲語。「させていただく」も謙譲語 → 二重謙譲。
```

#### ❌ No.5：「いただけますでしょうか」（過剰敬語）
```
△「ご確認いただけますでしょうか。」（丁寧すぎで不自然）
✅「ご確認いただけますか。」
✅「ご確認のほどよろしくお願いいたします。」（書面では〇）
```

#### ❌ No.6：お客様に「ご苦労様」
```
❌「お客様、長らくお待たせしてご苦労様でした。」
✅「お客様、大変お待たせいたしました。」
```

#### ❌ No.7：「〜のほうは」の多用
```
❌「コーヒーのほうはいかがでしょうか。」
✅「コーヒーはいかがでしょうか。」
→「〜のほう」は方向を示す言葉。明確な比較対象なく使うのは不適切。
```

#### ❌ No.8：自社の上司に敬語（ウチ・ソト混同）
```
❌（取引先に）「田中部長がいらっしゃいます。」
✅「田中はおります。」（ウチの人物はソトに対して低める）
```

#### ❌ No.9：「参考になりました」を目上に使う
```
❌「部長のお話、とても参考になりました。」
✅「部長のお話、大変勉強になりました。」
→「参考にする」は対等以下の情報に使う言葉。目上には「勉強になる」。
```

#### ❌ No.10：「よろしかったでしょうか」（過去形の誤用）
```
❌「ご注文は〇〇でよろしかったでしょうか。」
✅「ご注文は〇〇でよろしいでしょうか。」
→ 今確認していることに過去形は不自然。
```

---

### ✏️ Final Exercises

**Exercise 8-A：誤用発見（BJT形式）**
次の会話で、誤用がある部分を1つ選んでください。

> 取引先の田中様と電話。自分：ABCシステムの佐藤。
>
> A.「お電話ありがとうございます。ABCシステムの佐藤と申します。」
> B.「山田部長はただいま席をお外しになっております。」
> C.「戻り次第、こちらよりご連絡いたします。」
> D.「よろしければ、ご用件を伺えますでしょうか。」

**解答**: B — 「山田（自社の上司）」に尊敬語「お外しになる」を使っている。ウチの人物をソト（取引先）の前で高めるのは誤り。正しくは「山田はただいま席を外しております。」
